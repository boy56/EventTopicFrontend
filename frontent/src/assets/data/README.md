#武汉大船前端页面数据

4个页面的json数据

#data.json 主页面数据：

'topic'：新闻部分数据
	'index'：为对应新闻的id，每个新闻内部为字典，有相应新闻字段

'exports'：观点部分数据
	'i'：新闻部分第i个新闻对应的观点，为list
	

'static'：热度部分数据
	'x':横坐标 日期
	'y':纵坐标 热度值

"event1"：事件分析部分数据，为list，每个元素对应新闻部分的一条
	第一个元素为：横坐标
	第二个元素为：纵坐标，即热度
	第4-7元素为：保留字段，暂时无意义

#xuanti_data.json 选题页面数据，view_data.json 观点页面数据

为list，每条表示选题页面的一条，每条为字典，字典key值表示不同含义，可添加字段

#eventa_data.json 事件分析页面数据，暂无，目前为页面死数据
