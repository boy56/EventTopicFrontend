# WuhanFrontend
#武汉大船前端项目

启动方式：
进入frontent输入：
cnpm install
cnpm run start

代码结构：
主页面位置：\WuhanFrontend\frontent\src\modules\App.vue

综合选题页面：\WuhanFrontend\frontent\src\modules\xuanti

专家观点页面：\WuhanFrontend\frontent\src\modules\view

事件分析页面：\WuhanFrontend\frontent\src\modules\eventa

数据位置：WuhanFrontend\frontent\src\assets\data 包含4个页面的json数据，具体内容在data中
